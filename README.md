# Test Login Page

This project is a simple login page created using **HTML** and **CSS**.  
It includes a navigation bar, login form, and three sample test cases (positive, negative username, and negative password).

## What I did
- Built a clean webpage layout using HTML.
- Created a top navigation bar with links.
- Designed a centered login form for username and password.
- Added three test case sections with ordered lists.
- Styled the page using CSS (alignment, spacing, fonts, and colors).
- Added hover underline effect for navbar links.
- Designed footer section similar to the reference website.

## Technologies Used
- HTML5  
- CSS3
